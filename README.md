# Frontend Proyecto tienda Pelayo Ayuso

Proyecto final del Bootcamp de Neoland Web Development.


## Descripción

Aplicacion web para gestionar un catálogo de productos.


## Tecnologías

- React
- Vite
- CSS
- JavaScript


## Clonar el repositorio

```shell
git clone <url_repo>
git switch dev
```


## Cómo ejecutar el proyecto

```shell
npm install
npm run dev
```
